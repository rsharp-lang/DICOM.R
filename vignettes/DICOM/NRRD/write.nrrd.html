write.nrrd {NRRD} R Documentation

create a nrrd file based on a given collection of the image data objects.

Description

Usage

write.nrrd(file, rasters);

Arguments

file

-

rasters

a collection of the image data objects. all of the raster object inside this given collection should be in the same dimension size!

env

[as Environment]

Details

the required `rasters` data collection element could be one of the: 1. the gdi+ Image data object 2. the RasterMatrix for do heatmap rendering for a collection with only one ratser object inside, 2d nrrd object will be generates, for a collection with multiple raster object inside, 3d nrrd object will be generates.

Authors

Value

this function returns data object of type any kind.
